기본
오픈/클로즈, 프로젝트 목록, 안내사항 등 텍스트로 이루어진 정보가 많은 경우
단, 1행으로만 이루어진 경우, Box는 고정 높이 최소 64px을 유지하며, 그 안에서 middle 정렬을 적용합니다.

Component
여러 component들을 조합하여 적용할 경우에 사용합니다.
기본적으로 8의 배수로 적용
영역 내에서 그룹핑이 필요한 경우 4배수로 분할하여 적용도 가능
고정 영역 내 나열하는 경우 : 상하좌우 여백을 제외한 나머지 영역에서 균등하게 분할하여 배치
다만, 일부 케이스에서는 8배수 단위로 사이간격을 적용하여 배치가 가능(Ex. New Project > 선택, 추가된 Project 리스트 영역)
※ 좀 더 세분화 된 간격값은 각 구성요소에서 더 확인 가능합니다.
